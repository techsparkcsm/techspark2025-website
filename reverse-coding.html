Reverse Coding

SRK TECHSPARK 2025 | Department of CSE – AI & ML

About the Event

Reverse Coding is a logic-based programming challenge where participants are given only the expected output, sample test cases, and constraints. The task is to analyze the output pattern and write a program that reproduces it.

This event tests analytical thinking, problem-solving skills, and programming accuracy under time pressure.

Rules & Guidelines

  1. Participants can take part individually.
  2. Only output, sample test cases, and constraints will be provided.
  3. Participants must identify the logic and write a program.
  4. Allowed languages: C, C++, Java, Python.
  5. Internet usage and electronic gadgets are strictly prohibited.
  6. Code must be written only on the provided system.
  7. Plagiarism or malpractice will lead to disqualification.
  8. In case of a tie, accuracy and submission time will be considered.
  9. The judges’ decision is final.
  10. Registration amount per person is 50 rupees.
