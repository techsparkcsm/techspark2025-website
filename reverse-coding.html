Reverse Coding

SRK TECHSPARK 2025 | CSM Department

About the Event

Reverse Coding is a logic-based programming challenge where participants are given only the expected output, sample test cases, and constraints. The task is to analyze the output pattern, identify the hidden logic, and write a program that reproduces the given output.

This event tests analytical thinking, problem-solving skills, and programming accuracy under time pressure.

Rules & Guidelines

  1. Participants can take part individually.
  2. Only output, sample test cases, and constraints will be provided — no direct logic.
  3. Participants must identify the logic and write a program to match the given output.
  4. Allowed languages: C, C++, Java, Python.
  5. Internet usage and electronic gadgets are strictly prohibited.
  6. Code must be written only on the provided system.
  7. Plagiarism or malpractice will lead to immediate disqualification.
  8. In case of a tie, the decision will be based on accuracy and submission time.
  9. The judges’ decision is final.
